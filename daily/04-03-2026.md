- What is database middleware?
	- Database middleware is database connectivity software through which application programs connect and communicate with data repositories.
- Which of the following is NOT a component of Open Database Connectivity (ODBC) architecture?
	- Dynamic-link library
- Which of the following is NOT a method to achieve database connectivity?
	- Incorrect. Vendor-independent SQL connectivity is not a method to achieve database connectivity because DBMS vendors provide their own methods for connecting to their databases.
- Which of the following is NOT a characteristic nor a benefit of Internet technologies?
	- Improved data security
- Which of the following best defines CGI?
	- Common Gateway Interface (CGI) is a web server interface standard that uses script files to perform specific functions based on a client's parameters.
- Which of the following is not an activity of a web application server?
	- Incorrect. Web application servers can be used to create dynamic, not static, web searches.
