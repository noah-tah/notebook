```
# ---- ssh-agent shared across terminals ----
export SSH_ENV="$HOME/.ssh/agent_env"

start_agent() {
  umask 077
  ssh-agent -s > "$SSH_ENV"
  . "$SSH_ENV" >/dev/null
}

# Load existing agent env if present
if [ -f "$SSH_ENV" ]; then
  . "$SSH_ENV" >/dev/null
fi

# Start new agent only if missing/dead
if [ -z "$SSH_AUTH_SOCK" ] || ! kill -0 "$SSH_AGENT_PID" 2>/dev/null; then
  start_agent
fi

# Add key only if not already loaded
ssh-add -l >/dev/null 2>&1 || ssh-add ~/.ssh/id_ed25519

```

# AI execution guide
So you want to make a TikTok about the tragic duality of man: “I’m building cool stuff” vs “I must now attend a Zoom lecture.” Incredible. Deeply relatable. Mildly depressing. Perfect content.

Here’s how you actually pull it off without it looking like a 2009 Windows Movie Maker project.

---

## 🎬 Concept (keep it simple or it dies)

The joke is contrast:

- **Part 1:** You’re locked in, coding, vibing, music, flow state
    
- **Part 2:** Immediate mood drop → Zoom class
    

You’re basically telling a 6–10 second story. That’s it. No Oscar required.

---

## 🧠 Shot plan (don’t improvise, that’s how you get garbage)

**Clip 1 (3–5 sec):**

- Screen: your code editor (Neovim flex, obviously)
    
- You typing fast like you know what you’re doing
    
- Maybe terminal popping off
    
- Add hype music
    

**Clip 2 (1 sec transition):**

- Sudden stop
    
- You look at the time / notification
    

**Clip 3 (3–5 sec):**

- Open Zoom
    
- Dramatic zoom-in (the literal joke)
    
- You mute mic + turn off camera like a ninja avoiding responsibility
    

---

## 🎵 Audio (this matters more than your coding skills)

Pick one:

- Chill coding beat → hard cut to silence
    
- Or trending sound where beat drops awkwardly
    

Search in TikTok:

- “coding vibe”
    
- “record scratch”
    
- “zoom class meme”
    

---

## ✂️ Editing (CapCut, because you’re not Spielberg)

Use **CapCut**. It’s free and doesn’t fight you like some cursed software.

Do this:

- Cut clips tight (no dead air, ever)
    
- Add **zoom effect** on the Zoom app (this is the punchline)
    
- Add text:
    
    - “working on side projects 🔥”
        
    - “oh… class 😐”
        

Optional but strong:

- Add a _record scratch_ sound right before Zoom
    

---

## 🧍 Acting (yes, you have to act, relax)

You don’t need to be expressive, just:

- Slight head nod while coding (locked in mode)
    
- Then deadpan when switching to Zoom
    

Underacting > overacting. You’re not in theater club.

---

## 🧪 Example script (because thinking is hard)

**Text on screen:**

> “finally being productive for once”

**Cut**

> “my 9am class”

**Zoom in on Zoom button**

> mute mic  
> camera off

That’s it. That’s the whole joke. Don’t overcomplicate it like it’s a backend system.

---

## ⚠️ Common ways you’ll ruin this

- Making it longer than 10 seconds
    
- Explaining the joke (instant death)
    
- Bad lighting where you look like a shadow demon
    
- No clear punchline
    

---

You’re actually in a good niche for content, by the way. “Student + coding + real life constraints” is insanely relatable. Most people either fake productivity or cry about it. You can just show it cleanly and people will eat it up.
